# Basic Banking Exercise

This exercise started as practice for Python's os module and file handling.

During development I explored:
- Creating directories with os.mkdir()
- Renaming files and folders
- Reading and writing text files
- Exception handling
- Menu-driven programs

Files:
- sample.py -> initial practice code
- test.py -> experiments and testing
- main.py -> final exercise implementation

Lessons learned:
- Difference between os.remove() and os.rmdir()
- How to persist data using files
- How to read data back into variables
- Input validation and error handling
