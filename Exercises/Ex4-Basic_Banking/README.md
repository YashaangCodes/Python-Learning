# Banking System

A simple banking system built using Python.

## Features

- Create an account
- Deposit money
- Withdraw money
- Check current balance
- View transaction history

## Technologies Used

- Python 3

## How to Run

1. Clone the repository

```bash
git clone <repository-url>
```

2. Navigate to the project folder

```bash
cd Banking-System
```

3. Run the program

```bash
python main.py
```

## Example

```text
1. Deposit
2. Withdraw
3. Check Balance
4. Transaction History
5. Exit
```
